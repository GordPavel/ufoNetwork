Система представляет из себя социальную сеть для сущностей с разных планет.

Пользователь имеет возможность искать и просматривать профили других пользователей и группы без регистрации/аутентификации в сети. Поиск может осуществляться как по какому-то одному параметру пользователя или группы, так и по нескольим сразу.

При регистрации пользователь должен использовать уникальный логин, а так же указать свою расу (выбрать из имеющихся или создать новую), планету (выбрать из имеющихся или указать новую), пол, возраст. Так же пользователь может загрузить png-файл, для использования в качестве аватара. Система присваивает каждому зарегестрированному пользователю уникальный ID, а так же фиксирует дату и время регистрации.
Аутентифицированный пользователь может редактировать информацию о себе, за исключением ID и времени регистрации, которые не доступны пользователю.

Аутентифицированный пользователь может состоять в неограниченном колличестве групп, а так же сам создавать различные группы, оставлять в них сообщения и удалять собственные сообщения.

Пользователь, создавший группу, может исключить из неё любого другого пользователя и удалить любое написанное в ней сообщение. 

При создании группы пользователь должен указать её название и выбрать категорию группы из имеющегося списка. Группа имеет уникальный идентификатор (ID), хранит ссылку на создавшего её пользователя и может хранить загруженный пользователем png файл, который может испльзоваться как аватар группы. П
ользователь, создавший группу, может изменить её название, категорию, аватар.
